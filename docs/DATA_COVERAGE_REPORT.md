## 🔧 数据处理方法

### 1. 统一评估期间
- **所有资产使用相同的评估期间**: 2015-01-01 至 2024-12-31
- **晚上市资产**: 从实际上市日期开始，但仍在同一评估框架内
- **最少观测数**: 2,361天（VTEB，约9.4年）

### 2. 不平衡面板处理
- **方法**: 采用不平衡面板设计
- **每对独立**: 每个source-target pair使用各自可用的共同样本期
- **最少共同样本**: 要求至少2,000个共同交易日
- **缺失值处理**: 前向填充（最多5天）

### 3. 共同样本期计算
对于每个source-target pair：
```
共同样本期 = max(source_start, target_start) 至 min(source_end, target_end)
```

### 4. 横截面可比性
- **统一评估期间**: 所有pair在2015-2024期间评估
- **标准化指标**: Sharpe比率已按各自波动率标准化
- **Bootstrap推断**: 考虑了不同样本量的不确定性

- 另一处写"January 2005 – January 2026"
- 摘要日期是July 2026

**修正**:
- 统一为"January 1, 2015 – December 31, 2024"
- 删除所有不一致的旧文本
